# Zepto Web Traffic & Conversion Analytics Dashboard

## Brief One-Line Summary
Built an interactive Tableau dashboard to analyze website traffic, user engagement, and conversion behavior, helping Zepto identify opportunities to improve customer acquisition and conversion rates.

---

# Overview

Zepto is one of India's fastest-growing quick-commerce platforms, serving millions of users through its website and mobile application. Despite attracting significant traffic, not all visitors complete a purchase.

This project focuses on understanding user behavior across traffic sources, devices, visitor segments, and engagement metrics to identify factors influencing conversion performance.

Using Tableau, an interactive dashboard and data story were developed to help marketing and growth teams monitor key KPIs, uncover conversion bottlenecks, and make data-driven decisions to improve business performance.

---

# Problem Statement

Zepto's marketing team wants to understand:

- Which traffic sources generate the highest quality users.
- How user engagement impacts conversion rates.
- Whether mobile and desktop users behave differently.
- Which visitor segments contribute most to conversions.
- How website performance changes over time.

The objective is to transform raw web traffic data into actionable business insights that can drive higher conversions, better marketing ROI, and improved user experience.

---

# Dataset

The dataset contains detailed website traffic and user engagement information, including:

| Column | Description |
|----------|------------|
| Date | Traffic date |
| Traffic Source | Direct, Paid, Organic, Referral etc. |
| New vs Returning Visitors | Visitor segmentation |
| Device Type | Mobile/Desktop |
| Browser Type | Browser used by visitors |
| Operating System | OS used by visitors |
| Sessions | Total user sessions |
| Page Views | Pages viewed per session |
| Bounce | Users leaving without meaningful interaction |
| Avg. Session Duration | Average time spent on website |
| Conversion | Transaction completed (Yes/No) |

# Key Insights

## Traffic Source Performance

- Organic and Direct traffic contributed the majority of sessions and conversions.
- Paid traffic generated significant traffic volume but showed lower conversion efficiency.

## User Engagement

- Higher session duration and page views were strongly associated with improved conversion rates.
- High bounce-rate segments consistently demonstrated lower conversion performance.

## Device Analysis

- Mobile users represented the majority of traffic.
- Desktop users generally exhibited higher engagement and conversion rates.

## Visitor Behavior

- Returning visitors converted at a significantly higher rate than first-time visitors.
- Repeat users demonstrated stronger engagement across key metrics.

## Conversion Funnel

- A noticeable drop-off was observed between website visits and completed transactions, highlighting optimization opportunities within the user journey.

# Results & Conclusion

The dashboard successfully provides a comprehensive view of Zepto's website performance by connecting traffic acquisition, user engagement, and conversion metrics.

The analysis revealed that:

- Traffic quality matters more than traffic volume.
- Returning users drive stronger business outcomes.
- Engagement metrics are leading indicators of conversion success.
- Certain acquisition channels require optimization to improve ROI.

These insights can help Zepto's marketing and growth teams make informed decisions regarding user acquisition, retention, and conversion optimization strategies.
