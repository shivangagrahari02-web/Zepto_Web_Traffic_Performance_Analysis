# Analysis Checkpoints

The project was executed through a structured analytics framework, progressing from business understanding to actionable insights.

### Checkpoint 1: Business Understanding
- Defined the primary objective of improving website conversion rates.
- Identified key stakeholders as Zepto's Marketing and Growth teams.
- Established core business questions around traffic quality, user engagement, and conversion performance.

### Checkpoint 2: Data Exploration & Validation
- Reviewed dataset structure and data completeness.
- Validated dimensions such as Traffic Source, Device Type, Visitor Type, Browser, and Operating System.
- Checked for missing values, duplicates, and inconsistencies across key metrics.

### Checkpoint 3: KPI Identification
- Defined business-critical KPIs:
  - Total Sessions
  - Conversion Rate
  - Bounce Rate
  - Average Session Duration
  - Page Views per Session
  - Total Conversions
- Established KPI relationships to understand the user journey.

### Checkpoint 4: User Segmentation Analysis
- Segmented traffic by:
  - Traffic Source
  - Device Type
  - New vs Returning Visitors
  - Browser Type
  - Operating System
- Evaluated performance differences across user groups.

### Checkpoint 5: Traffic Source Performance Analysis
- Assessed contribution of Direct, Organic, Paid, and Referral traffic.
- Compared traffic volume against conversion efficiency.
- Identified high-volume and high-converting acquisition channels.

### Checkpoint 6: User Engagement Analysis
- Analyzed the relationship between:
  - Session Duration
  - Page Views
  - Bounce Rate
  - Conversion Rate
- Identified engagement patterns associated with successful conversions.

### Checkpoint 7: Device Performance Analysis
- Compared Mobile and Desktop user behavior.
- Evaluated differences in:
  - Sessions
  - Bounce Rate
  - Engagement
  - Conversion Rate
- Identified potential optimization opportunities by device category.

### Checkpoint 8: Conversion Funnel Analysis
- Tracked user progression from:
  - Website Visit
  - User Engagement
  - Conversion
- Measured drop-off points within the customer journey.
- Highlighted potential friction areas impacting conversion performance.

### Checkpoint 9: Trend Analysis
- Evaluated daily and periodic traffic patterns.
- Identified fluctuations in sessions, engagement, and conversions over time.
- Detected performance spikes and declines requiring business attention.

### Checkpoint 10: Dashboard Development & Storytelling
- Designed interactive KPI cards for executive monitoring.
- Created drill-down visualizations for detailed analysis.
- Integrated dynamic filters for self-service exploration.
- Structured Tableau Story to guide users from traffic acquisition to conversion insights.

### Checkpoint 11: Business Recommendations
- Prioritize high-converting traffic channels for acquisition investment.
- Improve landing page experience for high-bounce segments.
- Increase retention initiatives targeting returning visitors.
- Optimize mobile user experience to reduce friction in the conversion journey.
- Continuously monitor engagement metrics as leading indicators of conversion performance.
